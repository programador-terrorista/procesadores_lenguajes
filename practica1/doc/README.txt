Compilador cpt.jar (V1.0)
------------------------------
Análisis léxico y sintáctico


Invocar como:

-------------------------------------------------------------
java -jar cpt.jar <fichero_fuente_cpt>
-------------------------------------------------------------

Alternativamente

-------------------------------------------------------------
java -jar cpt.jar <fichero_fuente_cpt.cpt>
-------------------------------------------------------------

Si se quieren obtener por salida estandar los tokens generados, escribir un argumento después del fichero fuente.

Características generales:
No genera código, detecta si el texto en el fichero fuente es sintáctica y léxicamente correcto
en el lenguaje CPT.
